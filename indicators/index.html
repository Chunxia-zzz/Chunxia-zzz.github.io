RSI — Relative Strength Index（相对强弱指标）
类型：动量指标 · 创始人：J. Welles Wilder（1978）· 常用参数：RSI(14)
定义

RSI 衡量一段时间内价格上涨力度与下跌力度的相对比值，将价格变动的速度和幅度压缩到 0–100 区间，用于判断资产是否处于超买或超卖状态。

核心思想：如果近期涨幅远大于跌幅 → RSI 趋近 100（过热）；如果跌幅远大于涨幅 → RSI 趋近 0（过冷）。

计算方法

第 1 步：计算每日涨跌幅

变动 = 今日收盘价 - 昨日收盘价 变动 > 0 → 归入"上涨" 变动 < 0 → 归入"下跌"（取绝对值）

第 2 步：分别求平均涨幅和平均跌幅（以 14 日为例）

// 首次计算（简单平均） AG = 过去14天中所有上涨日涨幅之和 / 14 AL = 过去14天中所有下跌日跌幅之和 / 14 // 后续计算（Wilder 指数平滑） AG = (前一日AG × 13 + 今日上涨幅度) / 14 AL = (前一日AL × 13 + 今日下跌幅度) / 14

第 3 步：计算 RS 和 RSI

RS = AG / AL RSI = 100 - (100 / (1 + RS))
直觉理解：14 天全涨无跌 → RSI = 100；全跌无涨 → RSI = 0；涨跌各半且幅度相当 → RSI ≈ 50。
判断方法

① 超买 / 超卖（基础用法）

RSI 区间状态含义
> 70超买上涨动能过热，短期有回调压力
> 80强烈超买极端过热，回调概率很高
30 – 70中性趋势正常运行区间
< 30超卖下跌动能过度释放，可能反弹
< 20强烈超卖极端超卖，反弹概率很高

② 背离信号（高级用法，更可靠）

看涨背离（底背离）

价格创新低，但 RSI 没有创新低（低点抬高）

含义：下跌动能在衰减，卖方力量枯竭，即将反转向上。

看跌背离（顶背离）

价格创新高，但 RSI 没有创新高（高点降低）

含义：上涨动能在衰减，买方力量不足，即将反转向下。

③ 中轴线（50）判断趋势方向

条件信号判断
RSI 持续在 50 以上运行多头趋势回踩 40–50 区域可视为支撑
RSI 持续在 50 以下运行空头趋势反弹至 50–60 区域可视为阻力
RSI 从下方突破 50趋势转多空转多信号
RSI 从上方跌破 50趋势转空多转空信号

④ 不同周期的权重

周期信号意义原因
周线 RSI < 30极强买入信号出现频率 <5%，噪音少，历史胜率高
日线 RSI < 30参考信号出现频率高，容易假信号，需配合其他指标
周线 + 日线同时 < 30多周期共振 — 最强信号大小周期同时超卖，反弹确定性极高
常见误区："RSI < 30 就买入"是错的。在强下跌趋势中，RSI 可以长时间维持在 20–30 区间，单凭超卖抄底会持续套牢。RSI 应与 MACD 背离、VIX、成交量等多指标共振使用。
2
MACD — Moving Average Convergence Divergence（指数平滑异同移动平均线）
类型：趋势 + 动量指标 · 创始人：Gerald Appel（1979）· 常用参数：MACD(12, 26, 9)
定义

MACD 通过两条不同周期的指数移动平均线（EMA）之间的差值来揭示价格趋势的方向、强度和转折点。

它由三个组件构成：MACD 线（快线与慢线之差）、信号线（MACD 线的平滑）、柱状图（MACD 线与信号线之差）。柱状图是最灵敏的部分，能最早反映动能变化。

计算方法

第 1 步：计算快慢 EMA

// EMA = 指数移动平均，给近期价格更高权重 // 乘数 = 2 / (周期 + 1) EMA(12) = 12日指数移动平均线 ← 快线，反应更灵敏 EMA(26) = 26日指数移动平均线 ← 慢线，反应更平缓

第 2 步：计算 MACD 线（DIF）

MACD线 (DIF) = EMA(12) - EMA(26) // DIF > 0：短期均线在长期均线上方 → 多头趋势 // DIF < 0：短期均线在长期均线下方 → 空头趋势

第 3 步：计算信号线（DEA）

信号线 (DEA) = MACD线的 9日 EMA // 对 MACD 线做二次平滑，用于产生交叉信号

第 4 步：计算柱状图（Histogram）

柱状图 = (MACD线 - 信号线) × 2 柱状图 > 0 且递增 → 上涨动能加速 柱状图 > 0 且递减 → 上涨动能衰减 柱状图 < 0 且递减（绝对值增大） → 下跌动能加速 柱状图 < 0 且递增（绝对值缩小） → 下跌动能衰减
判断方法

① 金叉 / 死叉（基础用法）

金叉 — 看涨

MACD 线从下方上穿信号线

若发生在零轴下方 → 更强的看涨信号（空转多）

死叉 — 看跌

MACD 线从上方下穿信号线

若发生在零轴上方 → 更强的看跌信号（多转空）

② 零轴判断大趋势

条件信号含义
MACD 线在零轴上方多头趋势EMA(12) > EMA(26)，短期强于长期
MACD 线在零轴下方空头趋势EMA(12) < EMA(26)，短期弱于长期
MACD 线上穿零轴趋势转多短期均线上穿长期均线
MACD 线下穿零轴趋势转空短期均线下穿长期均线

③ 柱状图背离（最有价值的用法）

柱状图看涨背离

价格创新低，但柱状图低点抬高（负柱缩小）

含义：下跌动能衰减，是领先信号，通常比价格反转早 1–5 根 K 线。

精确定义：需连续 2 根以上柱线缩小，且低点高于前次低点。

柱状图看跌背离

价格创新高，但柱状图高点降低（正柱缩小）

含义：上涨动能衰减，预示顶部可能形成。

④ 多周期用法

周期用途说明
日线 MACD 柱状图背离入场时机捕捉短期反转点，定位精确买卖点
周线 MACD 柱状图缩小大趋势确认大周期卖方动能衰减，与日线背离共振 = 多周期反转前兆
注意：MACD 是滞后指标（基于均线），在震荡市中容易产生频繁的假金叉/死叉。柱状图背离是其中最领先的用法，但也需配合 RSI、成交量等确认。
3
Bollinger Bands — 布林带
类型：波动率指标 · 创始人：John Bollinger（1980s）· 常用参数：(20, 2)
定义

布林带由三条线组成，以移动平均线为中轴，上下各加减一定倍数的标准差，形成一个随波动率自动伸缩的"价格通道"。

核心思想：价格大部分时间在通道内运行（约 95%），触及通道边缘意味着价格在统计上处于极端位置。

计算方法
// 以 (20, 2) 参数为例 中轨 = 20日简单移动平均线 SMA(20) 上轨 = 中轨 + 2 × 标准差(20日) 下轨 = 中轨 - 2 × 标准差(20日) // 标准差 σ = √(Σ(收盘价 - 均值)² / N) // 通道宽度 = 上轨 - 下轨 // 波动率高 → 通道变宽（开口） // 波动率低 → 通道变窄（收口）
统计学含义：在正态分布假设下，±2σ 覆盖约 95.4% 的数据。意味着价格触及上轨/下轨的概率各约为 2.3%，属于统计上的"极端事件"。
判断方法

① 触轨信号

条件信号含义
价格触及或跌破下轨潜在超卖统计上约 2.5% 的 K 线触及下轨，具备均值回归基础
价格触及或突破上轨潜在超买短期过热，可能回归中轨
价格沿上轨持续运行强势上涨"走布林带上轨"是强趋势特征，不宜做空
价格沿下轨持续运行强势下跌"走布林带下轨"是弱势特征，不宜抄底

② 收口与开口（Squeeze）

收口 → 开口（Squeeze 突破）

通道极度收窄后突然向上开口 → 波动率爆发，大概率开启一波趋势行情。

收口期是暴风雨前的平静，方向由突破方向决定。

开口过大后收窄

通道极度张开后开始收窄 → 极端行情结束，趋势减速，价格可能回归中轨。

③ 中轨的支撑/阻力作用

趋势中轨作用说明
上升趋势中动态支撑回踩中轨（SMA20）后反弹，是加仓点
下降趋势中动态阻力反弹至中轨附近受阻回落
常见误区："价格触下轨就买入"是危险的。在单边下跌趋势中，价格可以沿下轨持续滑落（"走布林带下轨"）。触轨只是提供均值回归的统计基础，必须配合其他指标（RSI 超卖、MACD 背离、缩量等）确认。
4
MA — Moving Average（移动平均线）
类型：趋势指标 · 最基础的技术分析工具 · 关键参数：MA20 / MA50 / MA200
定义

移动平均线是将过去 N 个周期的收盘价取平均值，连成一条平滑曲线，用于过滤价格噪音、识别趋势方向。

周期越长，曲线越平滑，对趋势变化的反应越慢但越可靠；周期越短，越灵敏但噪音越多。

计算方法

SMA（简单移动平均）

SMA(N) = (C₁ + C₂ + ... + Cₙ) / N // C = 收盘价，N = 周期 // 所有价格权重相等

EMA（指数移动平均）

乘数 k = 2 / (N + 1) EMA(今日) = 今日收盘价 × k + 昨日EMA × (1 - k) // 近期价格权重更高，反应更灵敏 // 例：EMA(12) 的乘数 = 2/13 ≈ 0.1538 // 例：EMA(26) 的乘数 = 2/27 ≈ 0.0741

关键均线周期及含义

MA5/MA10超短期趋势，适合日内/短线交易者
MA20≈ 1 个月交易日，短期趋势参考（= 布林带中轨）
MA50≈ 1 个季度，中期趋势分水岭
MA200≈ 1 年，最重要的长期趋势线，机构和算法交易广泛参考
判断方法

① 价格与均线的位置关系

条件信号说明
价格在 MA200 上方长期多头趋势牛市环境，回调至 MA200 附近是支撑
价格在 MA200 下方长期空头趋势熊市环境，反弹至 MA200 附近是阻力
价格偏离 MA200 超过 8%–10%均值回归机会过度偏离长期均值，中长期有回归动力

② 金叉 / 死叉

金叉（Golden Cross）

MA50 上穿 MA200

经典长期看涨信号。历史上标普 500 出现金叉后 1 年平均涨幅约 15%。

死叉（Death Cross）

MA50 下穿 MA200

长期看跌信号。但有较多假信号（滞后性），需结合量能确认。

③ 均线排列

排列形态含义
多头排列MA5 > MA20 > MA50 > MA200，全部向上强势上涨趋势，顺势做多
空头排列MA5 < MA20 < MA50 < MA200，全部向下强势下跌趋势，不宜抄底
缠绕多条均线交织在一起震荡市/方向不明，等待突破
MA200 的特殊地位：MA200 被称为"牛熊分界线"。华尔街机构、对冲基金、量化策略广泛以 MA200 作为长期趋势判断依据。价格向 MA200 的回归是历史上最稳定的统计规律之一。
5
VIX — CBOE Volatility Index（恐慌指数）
类型：波动率 / 情绪指标 · 发布机构：CBOE（芝加哥期权交易所）· 基于标普 500 期权定价
定义

VIX 衡量市场对未来 30 天标普 500 波动率的预期，由标普 500 指数期权的隐含波动率推算而来。

它不是直接测量恐惧，而是测量市场参与者愿意为"保险"（看跌期权）支付多少溢价。恐慌时期大家抢购保险 → 期权价格飙升 → VIX 飙升。因此被称为"恐慌指数"。

计算方法（简化说明）
// VIX 的完整计算非常复杂，涉及期权定价模型 // 核心原理如下： 1. 收集标普500指数所有近月和次近月的 虚值看涨 + 虚值看跌期权 价格 2. 通过期权价格反推出 隐含波动率 3. 加权平均得到 30天预期年化波动率 // VIX = 20 的含义： // 市场预期未来30天标普500年化波动率为20% // 换算为月度：20% / √12 ≈ 5.8%，即预期月涨跌幅约 ±5.8%
判断方法

① VIX 绝对水平

VIX 区间市场状态含义
< 15低波动 / 平静市场自满，可能在酝酿波动（暴风雨前的平静）
15 – 20正常波动市场情绪平稳，趋势健康运行
20 – 30高波动市场焦虑加剧，波动加大
> 30恐慌有统计意义的极端恐慌，历史上是买入信号
> 40极端恐慌罕见事件（金融危机、疫情级别），反向买入胜率极高

② VIX 的反向指标特性

VIX 飙升 → 看涨（反向）

VIX 急剧飙升至 30+ 通常对应市场的恐慌性抛售底部。

统计数据：VIX 从 >40 回落后，标普 500 未来 12 个月全部为正收益。

VIX 极低 → 警惕（反向）

VIX 长时间维持在 <12 表示市场过度自满。

历史上极低 VIX 之后往往跟随波动率爆发。

③ VIX 期限结构

形态含义信号
正常升水（Contango）远月 VIX > 近月 VIX正常状态，市场对未来不确定性有合理预期
贴水（Backwardation）近月 VIX > 远月 VIX短期恐慌极端，但市场预期恐慌将消退 → 底部信号
注意：VIX 在持续性熊市中可以长期维持在 25–35 区间（如 2008 年长达数月 >40）。单独看 VIX 绝对值不足以判断底部，需配合 RSI、MACD 背离、成交量等共振确认。VIX > 30 只说明"恐慌已出现"，不说明"恐慌已结束"。
6
Volume Analysis — 成交量分析
类型：成交量指标 · "量在价先" — 成交量是价格变动的燃料
定义

成交量是指在特定时间周期内买卖双方成交的总股数（或合约数）。它反映市场参与者的活跃程度和交易意愿。

核心原则："量在价先"。价格的变动需要成交量的配合才具有可信度。没有量的支撑，价格变动往往不可持续。

关键概念
放量成交量显著高于近期平均水平（通常 >1.5x 或 >2x 20日均量）
缩量成交量明显低于近期平均水平
天量成交量达到阶段性极值，通常出现在趋势末期
地量成交量萎缩至极低水平，"地量见地价"，但需确认
均量线成交量的移动平均，常用 5 日均量和 20 日均量作为参照
判断方法

① 量价配合（健康趋势）

价格成交量含义
上涨放量量价齐升 — 上涨趋势健康，买方积极
上涨缩量量价背离 — 上涨动能不足，警惕回调
下跌放量恐慌性抛售 — 卖方主导，可能加速下跌
下跌缩量卖压减轻 — 抛售意愿降低，可能接近底部

② 恐慌放量 → 缩量企稳（底部特征）

经典底部形态：投降式抛售 + 缩量企稳

第 1 阶段：恐慌放量（Capitulation）

成交量达到阶段性天量，价格大幅下跌。这代表"最后的卖家"在恐慌中不计成本抛售 — 即"投降"。

第 2 阶段：缩量企稳

随后几日成交量急剧萎缩，但价格不再创新低。这意味着卖方力量衰竭，想卖的人已经卖完了。

第 3 阶段：放量上涨确认

如果随后出现放量上涨（突破前期阻力），则底部确认。

③ 天量信号

场景信号含义
上涨途中出现天量天量天价可能是最后一波疯狂买入，见顶信号
下跌途中出现天量恐慌投降卖方能量集中释放，可能是底部区域
实用技巧：单日成交量不看绝对值，而是看相对值（与 20 日均量的比值）。放量通常定义为 >1.5 倍均量，天量为 >3 倍均量。
7
CNN Fear & Greed Index（恐惧与贪婪指数）
类型：复合情绪指标 · 发布机构：CNN Business · 实时更新
定义

CNN 恐惧与贪婪指数是一个综合 7 个市场子指标的复合情绪指数，将市场情绪量化为 0–100 的单一数值。

核心逻辑："别人恐惧时贪婪，别人贪婪时恐惧"（巴菲特）。该指数帮助投资者客观量化当前市场的情绪温度。

计算方法 — 7 个子指标

CNN F&G 指数等权重综合以下 7 个市场信号，每个信号独立判断看涨/看跌/中性：

#子指标衡量内容
1Stock Price Momentum标普 500 相对其 125 日均线的位置
2Stock Price StrengthNYSE 创 52 周新高 vs 新低的股票数量
3Stock Price BreadthMcClellan Volume Summation Index（上涨 vs 下跌成交量）
4Put/Call Ratio看跌期权 vs 看涨期权的成交量比值
5Market Volatility (VIX)VIX 相对其 50 日均线的位置
6Safe Haven Demand国债 vs 股票的相对收益率（资金避险偏好）

判断方法

① 情绪区间

数值区间情绪状态操作参考
0 – 25极度恐惧 Extreme Fear反向买入信号 — 历史上 <10 后 1 年全部正收益
25 – 45恐惧 Fear市场偏悲观，开始关注买入机会
45 – 55中性 Neutral市场情绪平衡
55 – 75贪婪 Greed市场偏乐观，注意控制仓位
75 – 100极度贪婪 Extreme Greed反向卖出信号 — 市场过热，高位风险加大

② 极端值的统计意义

F&G < 25（极度恐惧）

7 个市场信号同时指向恐慌，是多维度共振的底部信号。

< 10 为历史极端，出现频率极低（如 2008 金融危机、2020 疫情底部），之后 1 年均为正收益。

F&G > 80（极度贪婪）

市场过度乐观，所有人都在买入。

虽然不意味着立即下跌（牛市末期可长期维持高位），但此时追高的风险收益比极差。

注意：F&G 指数是辅助确认工具，不宜单独使用。它的价值在于将 7 个不同维度的市场信号浓缩为单一读数，帮助你快速判断"市场情绪温度"。最佳用法是与技术指标（RSI、MACD）配合使用。

单一指标都有缺陷，多指标共振才能提高信号可靠性。
