Neocloud 是什么？

先澄清一个常见误解：Neocloud ≠ 云服务器的升级版。

一句话区分：

传统云（AWS/Azure/GCP） = 什么都卖的”IT超市”
Neocloud = 只卖 GPU 算力的”AI 军火商”

Neocloud（新云）是一类专门为 AI/ML 工作负载构建的 GPU 云基础设施提供商。它们不提供传统云的数百种服务，而是将所有资源集中在一件事上：为 AI 公司提供大规模 GPU 算力。

Neocloud vs 传统云：核心区别

维度 传统云 (AWS, Azure, GCP) Neocloud (CRWV, NBIS, APLD)
服务范围 数百种服务（计算、存储、数据库、AI、IoT…） 极窄：GPU 集群租赁 + AI 训练/推理
硬件核心 CPU 为主，GPU 是众多服务之一 全部围绕 NVIDIA GPU 建设
网络架构 通用云网络 InfiniBand / NVLink 高带宽互联（专为分布式训练优化）
散热方案 标准数据中心风冷 液冷 + 高密度电力基础设施
客户数量 数百万客户 少数大客户（AI 实验室、大厂溢出需求）
合同模式 按需付费 / 年度承诺 多年锁定大合同（动辄数十亿美元）
营收规模 $800-1000亿/年 $3-50亿/年（但增速惊人）
资金来源 经营现金流自给自足 疯狂举债买 GPU
存在逻辑 企业 IT 基础设施的底座 填补大厂 GPU 产能缺口

为什么 Neocloud 能存在？

逻辑很简单：

大厂自己建 GPU 集群的速度，跟不上 AI 需求的增长速度。

微软想给 OpenAI 提供算力 → 自己建来不及 → 从 CoreWeave 租 GPU。
Meta 要训练下一代 LLaMA → 自有产能不够 → 签 CoreWeave $210 亿合同。

Neocloud 本质上是 AI 军备竞赛中的产能缓冲层。它们的存亡，取决于这个产能缺口是否持续存在。

各标的逐一解析

CRWV（CoreWeave）— Neocloud 绝对龙头

背景： 2025年3月在纳斯达克 IPO，发行价 $40。

CoreWeave 是体量最大的 Neocloud，营收 $51.3亿，手握恐怖级别的客户合同：

  • Microsoft — 历史第一大客户（曾贡献 60%+ 收入）
  • Meta — $210亿合同（2026年4月签署）
  • Anthropic — 数十亿美元合同
  • Jane Street — $60亿 AI 云合同
  • OpenAI、NVIDIA — 均为其客户

亮点： 毛利率 71.7%，营收增速 168%，客户阵容无可匹敌。

风险点： 负债 $298亿，杠杆率 8.9 倍，利息一年吃掉 $12.3亿。尽管毛利率高达 72%，仍然巨亏 $23.6亿。本质上是一个用债务杠杆赌 AI 不减速的公司。

NBIS（Nebius）— 前 Yandex 国际业务拆分

背景： 从俄罗斯 Yandex 拆分出的国际 AI 基础设施公司，总部在荷兰阿姆斯特丹。

  • 营收增速最快（+351%），且已经盈利（净利 $1亿）
  • 杠杆率合理（1.1x），财务状况远比 CRWV 健康
  • 业务覆盖欧洲、北美、以色列
  • 旗下还有自动驾驶（Avride）、数据标注（Toloka）、分析数据库（ClickHouse）等业务
  • P/E 411 倍，市场在定价巨大的未来增长预期

APLD（Applied Digital）— AI 数据中心”地产商”

背景： 不直接运营 GPU 云，而是建造并出租高密度 AI 数据中心给大厂。

  • 100 MW 已上线，900 MW 在建，管线 3.5 GW+
  • 签了一个 $50亿 AI Factory 租约（客户为美国投资级大厂）
  • 拥有专利无水冷却技术
  • 体量最小，更像是 Neocloud 的上游——“卖水人”

Neocloud 的资本结构：借钱买 GPU 的游戏

Neocloud 是科技行业中最资本密集的商业模式之一：

公司 总负债 现金 股东权益 负债/权益
CoreWeave $298亿 $31亿 $33亿 8.9x
Nebius $49亿 $37亿 $46亿 1.1x
Applied Digital $28亿 $17亿 $26亿 1.1x

融资方式：

  • GPU 抵押贷款 — CoreWeave 开创了用 GPU 资产做抵押的融资模式
  • IPO 和增发 — CoreWeave 2025年3月 IPO 募资
  • 客户预付/长期合同 — 多年锁定合同提供现金流可见性，支撑进一步借债
  • 战略投资 — NVIDIA 是多家 Neocloud 的投资人

核心风险：不可忽视

Neocloud 的风险结构和传统云公司完全不同，必须充分认知：

1. 极致杠杆风险

CoreWeave 借了 $298亿买 GPU，每年光利息就 $12.3亿。AI 支出一旦放缓，现金流断裂的风险真实存在。

2. GPU 快速贬值

NVIDIA 每 1-2 年推出新架构（H100 → B200 → Rubin）。今天花重金购入的 GPU 集群，两年后可能大幅贬值，而债务不会消失。

3. 客户高度集中

CoreWeave 历史上 60%+ 收入来自微软一家。虽然正在分散化（Meta、Anthropic 等），但一个大客户砍单就是灾难。

4. 大厂自建的威胁

一旦 AWS / Azure / GCP 自有 GPU 产能跟上需求，Neocloud 的”产能缺口”逻辑就不再成立。这是最大的结构性风险。

5. AI 效率提升的风险

如果 AI 模型变得更高效（更小的模型、更少的算力需求），GPU 需求可能见顶。

怎么看 Neocloud 这个赛道？

角度 判断
赛道定位 不是传统云的竞争对手，是 AI 基建周期的产能填充者
类比 淘金热中”卖铲子的人”——但铲子是借钱买的
核心变量 AI capex 是否持续增长 → 是则继续爆发，否则杠杆反噬
龙头选择 CRWV 体量最大但杠杆最恐怖；NBIS 财务更健康且已盈利
配置建议 高弹性高风险品种，不适合重仓，适合作为 AI 主题的卫星仓位

龙头排序

如果在标的池里加 Neocloud 敞口，CRWV + NBIS 各放一个就够了。但务必记住这类标的的波动性极大（CRWV 52周从 $33 到 $187），仓位控制比选股更重要。
